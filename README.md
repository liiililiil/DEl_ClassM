# ClassM 제거 프로그램


* 사용법

1. Mclass 정지를 실행해주세요!


***
### 기능들
* 실행했을때 ClassM프로그램이 켜저있지 않으면, 자동으로 즉시 종료합니다.
* Mclass는 지속적으로 프로그램을 재시작합니다. 만약 재시작을 하지 않으면, 프로그램이 자동으로 종료됩니다.
